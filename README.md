# Система управления пользователями – Модуль 02

## Описание
В данном проекте реализована простая система управления пользователями
для небольшой компании.

Система позволяет:
- добавлять пользователей
- удалять пользователей
- обновлять данные пользователей

Каждый пользователь содержит имя, email и роль.

## Принципы проектирования
В проекте применены следующие принципы:
- YAGNI (You Aren’t Gonna Need It)
- KISS (Keep It Simple)
- DRY (Don’t Repeat Yourself)

## Структура проекта
- User – модель пользователя
- UserManager – управление коллекцией пользователей

## Технологии
- C#
- .NET
